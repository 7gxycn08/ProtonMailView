# ProtonMailView
Free and Private ProtonMail Windows Desktop App made with Electron.


# Main Features
1. Runs in tray.
2. Can start hidden in tray.
3. Can start at boot.
4. Supports Desktop Notifications.
5. Supports file attachment downloads.
